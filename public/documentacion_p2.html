METODOLOGÍAS DE DESARROLLO DE SW

¿Qué es una metodología de desarrollo de software?

Las metodologías de desarrollo de software son un conjunto de técnicas y métodos organizativos que se aplican para diseñar soluciones de software informático. El objetivo es organizar los equipos de trabajo para desarrollar las funciones de un programa de la mejor manera posible.
